## Usage

1. Install Node.js if not already installed.
2. Install Artillery globally:
   ```
   npm install -g artillery
   ```
3. Create or modify your test.yml as needed.
4. Run the test:
   ```
   artillery run test.yml
   ```
5. Generate a report:
   ```
   artillery run test.yml --output results.json
   ```
